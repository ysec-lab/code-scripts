# Notepad++ Tips & Tricks

A quick reference guide for essential Notepad++ line operations, view configurations, and plugin features.

## 🔀 Text & Line Operations

* **Remove Duplicate Lines:** `Edit` > `Line Operations` > `Remove Duplicate Lines`
* **Duplicate Lines:** `Edit` > `Line Operations` > `Duplicate Lines`
* **Remove Empty Lines:** `Edit` > `Line Operations` > `Remove Empty Lines`
* **Change Text Case (Upper/Lower):** Select the text/line > `Right-Click` > Select `Upper/Lower case`
* **Proper Case:** `Edit` > `Convert to Case` > `Proper Case`

## 🖥️ Workspace & View Customization

* **Dual View (Side-by-Side):** `Right-Click` on the document tab > `Move Document` > `Move to the Other View` (opens the file in another screen)
* **Clone View:** `Right-Click` on the document tab > `Move Document` > `Clone to the Other View` (opens the *same* file cloned in two screens)
* **Changing Themes:** `Settings` > `Style Configurator` > Select theme (e.g., zenburn). 
  * *Tip: You can customize this setting by choosing specific background and foreground colors.*
* **Enable Dark/Light Mode:** `Settings` > `Preferences` > `Dark Mode` > Toggle `Light Mode` or `Dark Mode`

## 🔒 Security & Hashes

* **Set File to Read-Only Mode:** `Right-Click` on the title of the document tab > Select `Read-Only`
* **Generate Hash for Text:** `Tools` > `MD5 to SHA512` > `Generate` > Paste your text to get the hash value
* **Generate Hash for File:** `Tools` > `MD5 to SHA512` > `Generate from files` > Select the file

## 🔌 Plugins

*Note: Access the plugin manager via `Plugins` > `Plugins Admin`*

### Note Taking
* **Install:** Search for a Note plugin in Plugins Admin > `Install`

### Markdown to PDF Conversion
1. **Install:** Search for `Markdown Viewer` in Plugins Admin > `Install`
2. **Preview:** Go to `Plugins` > `Markdown Viewer` to see the preview
3. **Export:** Click on `Export` within the viewer > Select `PDF` or `HTML`

### Compare Files
1. **Install:** Search for `Compare` in Plugins Admin > `Install`
2. **Usage:** Make sure the files are next to each other or open in tabs > Go to `Plugins` > `Compare`
3. *Tip: Turn on the NavBar for easier scrolling through differences.*
